# Walk Safe Cam

یک اپ ساده اندروید که با فشردن دکمه، یک نوار از تصویر زنده‌ی دوربین پشت (تلاش می‌کند لنز واید/اولترا‌واید را انتخاب کند) در **بالای صفحه**، روی همه‌ی برنامه‌ها، نمایش می‌دهد؛ بقیه صفحه مثل همیشه قابل استفاده است. هدف: دیدن مسیر جلوی پا هنگام راه رفتن و استفاده از گوشی.

## ساختار پروژه
- `MainActivity` — گرفتن دسترسی دوربین و دسترسی «نمایش روی برنامه‌های دیگر»، و تنظیم ارتفاع نوار.
- `OverlayService` — یک Foreground Service که با `WindowManager` یک پنجره‌ی شناور بالای صفحه اضافه می‌کند و با CameraX تصویر دوربین را در آن نمایش می‌دهد.

## نحوه بیلد گرفتن

### روش ۱: خودکار با GitHub Actions
1. این پوشه را در یک ریپوی جدید گیت‌هاب پوش کنید (شاخهٔ `main`).
2. فایل `.github/workflows/build.yml` به‌صورت خودکار پروژه را می‌سازد.
3. بعد از اتمام Action، از تب **Actions** روی اجرای موفق کلیک کنید و APK را از قسمت **Artifacts** دانلود کنید (`app-debug`).

### روش ۲: لوکال با Android Studio
1. پروژه را در Android Studio باز کنید (Gradle Sync خودش انجام می‌شود).
2. Run ▶️ روی گوشی یا شبیه‌ساز.

## نکات مهم استفاده
- اولین بار که «فعال کردن» را می‌زنید، دسترسی دوربین و سپس دسترسی «Display over other apps» را باید تأیید کنید.
- روی اندروید ۱۰ به بالا چون از دوربین در پس‌زمینه استفاده می‌شود، یک نوتیفیکیشن دائمی («Foreground Service») نمایش داده می‌شود؛ این الزام سیستم‌عامل است و قابل حذف نیست.
- انتخاب لنز واید صرفاً یک حدس مبتنی بر کوتاه‌ترین فاصلهٔ کانونی دوربین‌های پشت گوشی است؛ روی گوشی‌هایی که چند دوربین پشت دارند معمولاً درست کار می‌کند، ولی اگر گوشی فقط یک دوربین پشت دارد همان استفاده می‌شود.
- برای بستن نوار دوربین، دکمه‌ی «X» گوشه‌ی نوار یا دکمه‌ی «خاموش کردن» در اپ را بزنید.

## توسعه‌های بعدی که می‌توان اضافه کرد
- قابلیت جابه‌جایی/تغییر اندازهٔ نوار با کشیدن (drag/resize) به‌جای فقط اسلایدر تنظیم اولیه.
- دکمه‌ی شروع خودکار سرویس هنگام راه‌اندازی گوشی.
- امکان شفافیت/opacity قابل تنظیم برای نوار.
